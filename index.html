1. Einleitung

Diese Datenschutzerklärung gilt für die Android-App „Pincer for OpenClaw“ (im Folgenden „die App“). Die App ist ein unabhängiger Third-Party-Client für die Claude API von Anthropic und steht in keiner Verbindung zu Anthropic.

2. Verantwortlicher

Verantwortlich für die Datenverarbeitung im Zusammenhang mit dieser App ist der Entwickler, erreichbar über die im Google Play Store angegebenen Kontaktdaten.

3. Welche Daten werden erhoben?

Die App selbst erhebt, speichert oder übermittelt keine personenbezogenen Daten an den Entwickler. Insbesondere:

API-Schlüssel: Ihr API-Schlüssel wird ausschließlich lokal auf Ihrem Gerät gespeichert und nur für die direkte Kommunikation mit der Claude API bzw. Ihrem konfigurierten Gateway-Server verwendet.

Chat-Verläufe: Alle Gesprächsdaten werden ausschließlich lokal auf Ihrem Gerät gespeichert.

Keine Analyse-Tools:strong> Die App verwendet keine Tracking- oder Analyse-Dienste.

4. Datenweitergabe an Dritte

Wenn Sie die App nutzen, werden Ihre Nachrichten direkt an die Claude API (Anthropic) oder an einen von Ihnen konfigurierten Gateway-Server gesendet. Die Verarbeitung dieser Daten unterliegt den jeweiligen Datenschutzbestimmungen von Anthropic bzw. des Gateway-Anbieters. Der Entwickler dieser App hat keinen Zugriff auf diese Kommunikation.

5. Datenspeicherung

Alle Daten (API-Schlüssel, Chatverläufe, Einstellungen) werden ausschließlich lokal auf Ihrem Gerät gespeichert. Durch Deinstallation der App werden alle lokal gespeicherten Daten gelöscht.

6. Berechtigungen

Internetzugang: Erforderlich für die Kommunikation mit der Claude API bzw. dem Gateway-Server.

Kamerazugriff (optional): Wird nur verwendet, wenn Sie Fotos für die Bilderkennung (Vision) an Claude senden möchten.

7. Ihre Rechte

Da die App keine personenbezogenen Daten an den Entwickler übermittelt, liegen dem Entwickler keine personenbezogenen Daten von Ihnen vor. Sie können jederzeit alle lokal gespeicherten Daten löschen, indem Sie die App deinstallieren oder die App-Daten in den Geräteeinstellungen löschen.

8. Änderungen dieser Datenschutzerklärung

Der Entwickler behält sich vor, diese Datenschutzerklärung bei Bedarf anzupassen. Die aktuelle Version ist stets unter dieser URL abrufbar.

9. Kontakt

Bei Fragen zum Datenschutz wenden Sie sich bitte über die im Google Play Store hinterlegten Kontaktdaten an den Entwickler.
