Datenschutzerklärung

Zuletzt aktualisiert: 27. Februar 2026

1. Einleitung

Diese Datenschutzerklärung gilt für die Android-App „Pincer for OpenClaw“ (im Folgenden „die App“). Die App ist ein unabhängiger Third-Party-Client für die Claude API von Anthropic und steht in keiner Verbindung zu Anthropic.

2. Verantwortlicher

Verantwortlich für die Datenverarbeitung im Zusammenhang mit dieser App ist der Entwickler, erreichbar über die im Google Play Store hinterlegten Kontaktdaten.

3. Welche Daten werden erhoben?

Die App selbst erhebt, speichert oder übermittelt keine personenbezogenen Daten an den Entwickler.

Alle Daten werden ausschließlich lokal auf Ihrem Gerät gespeichert:

API-Schlüssel: Wird lokal gespeichert und nur zur Authentifizierung gegenüber der Claude API oder einem Gateway-Server verwendet.

Chatverläufe: Werden lokal in einer Datenbank auf Ihrem Gerät gespeichert.

Keine Analyse-Tools: Die App verwendet keine Tracking- oder Analyse-Dienste.

4. Datenweitergabe an Dritte

Wenn Sie die App nutzen, werden Ihre Nachrichten direkt an die Claude API (Anthropic) oder an einen von Ihnen konfigurierten Gateway-Server gesendet. Diese Kommunikation unterliegt den jeweiligen Datenschutzrichtlinien des API-Anbieters.

5. Datenspeicherung

Alle Daten (API-Schlüssel, Chatverläufe, Einstellungen) werden ausschließlich lokal auf Ihrem Gerät gespeichert. Durch Deinstallation der App werden alle lokalen Daten gelöscht.

6. Berechtigungen

Internetzugang: Erforderlich für die Kommunikation mit der Claude API bzw. dem Gateway-Server.

Kamerazugriff (optional): Wird nur verwendet, wenn Sie Fotos für die Bilderkennung (Vision) an Claude senden möchten.

7. Ihre Rechte

Da die App keine personenbezogenen Daten an den Entwickler übermittelt, liegen dem Entwickler keine personenbezogenen Daten von Ihnen vor. Sie können jederzeit alle lokal gespeicherten Daten durch Deinstallation der App löschen.

Der Entwickler behält sich vor, diese Datenschutzerklärung bei Bedarf anzupassen. Die aktuelle Version ist stets unter dieser URL abrufbar.

9. Kontakt

Bei Fragen zum Datenschutz wenden Sie sich bitte über die im Google Play Store hinterlegten Kontaktdaten an den Entwickler.
